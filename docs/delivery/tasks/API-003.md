## Objective

Convert the approved C ABI spike into a production-quality Zig adapter.

## Procedure

1. Wrap client creation, address parsing, submission, callback completion, five-second request timeout, ten-second cancellation/shutdown, and cleanup.
2. Keep every C pointer, context, and buffer alive until its callback finishes.
3. Serialize or coordinate operations according to the native client contract.
4. Implement a harmless real lookup health probe.
5. Map native failures to stable sanitized internal codes.
6. Cancel and shut down within ten seconds while remaining safe during in-flight work.

## Acceptance evidence

- Native leak/sanitizer evidence where available.
- Error-redaction scan.
- `zig build test` succeeds through API-001's static aggregate registration; `git diff --check` passes.
