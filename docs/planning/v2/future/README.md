# Deferred Native Platform Work

This directory preserves intent that is explicitly outside the current
Windows 11 x64 slice. These files are not registered delivery task guides and
do not confer native execution evidence, packaging support, or completion
credit.

Native macOS Intel, macOS Apple Silicon, and Linux work may return only through
a future ADR and task wave. That wave must revalidate current dependency pins,
native lifecycle behavior, platform containment, package layouts, provenance,
and smoke evidence on native workers.
