# Voyage VII

Voyage VII is a local-first desktop application. The current v2 slice is
supported only on Windows 11 x64. Native macOS and Linux packages are deferred
under [future planning](docs/planning/v2/future/README.md) and are not current
setup or support targets.

Start with the verified runbooks:

- [Development](docs/development/README.md)
- [Operations](docs/operations/README.md)
- [Packaging](docs/packaging/README.md)
- [Database schema documentation](docs/database/README.md)
- [Architecture decision records](docs/adr/README.md)

The product version is `0.1.0`, read from [VERSION](VERSION). The Windows
portable ZIP name is `voyage-vii_0.1.0_windows-x86_64.zip`.

This slice does not include financial product behavior, cloud deployment,
installers, production signing, notarization, auto-update, automatic reset,
repair, upgrade, or automatic backup tooling.
